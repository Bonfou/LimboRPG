    Weapon Shells

    W values

    In this document, wherever you see W, it represents the number of weapons of that shell you're wielding. For example, if you're wielding two handguns, W is equal to 2 for all Handgun abilities.

    Melee Weapon Shells

    Unarmed

    1-Handed

    Bare fists, brass knuckles, and heavy tails.

    FEA Feature: Flow State

    The first W times you generate EP during your turn, generate 1 additional EP.

    OFF Jab

    DIE: 3 // PWR: 2 // DMG: 5 // EP: 0

    Flow 1.

    OFF Haymaker

    DIE: 1 // PWR: 4 // DMG: 25 // EP: 2

    No additional effects.

    DEF Brace

    DIE: 4 // PWR: 1 // DMG: 0 // EP: 0

    On Break: This Move gains Flow 2.

    Blades

    1-Handed

    Knives, swords, axes, and daggers.

    FEA Feature: Lacerate

    The first W times you score a Win in a Clash each round, apply 2 stacks of Bleed.

    OFF Quick Slash

    DIE: 5 // PWR: 1 // DMG: 3 // EP: 0

    Afflict Bleed 1.

    OFF Cleave

    DIE: 2 // PWR: 3 // DMG: 9 // EP: 1

    Bulwark 3.

    DEF Riposte

    DIE: 1 // PWR: 5 // DMG: 5 // EP: 1

    Vicious.

    Bludgeons

    1-Handed

    Bats, pipes, maces, and blackjacks.

    FEA Feature: Crush Defense

    When you score a Break, apply 3 x W stacks of Tremor.

    OFF Smash

    DIE: 3 // PWR: 3 // DMG: 6 // EP: 0

    Flow 1.

    OFF Shattering Blow

    DIE: 1 // PWR: 5 // DMG: 16 // EP: 2

    Afflict Tremor 3.

    DEF Guard

    DIE: 3 // PWR: 3 // DMG: 0 // EP: 0

    No additional effects.

    Heavy Weapons

    2-Handed

    Sledgehammers, power fists, and greatswords.

    FEA Feature: Unstoppable

    Your Moves cannot be Broken on their first W dice.

    OFF Sunder

    DIE: 2 // PWR: 4 // DMG: 8 // EP: 1

    Flow 1, Bulwark 1.

    OFF Wide Swing

    DIE: 2 // PWR: 4 // DMG: 7 // EP: 2

    Area 1.

    DEF Heft

    DIE: 4 // PWR: 2 // DMG: 0 // EP: 0

    Bulwark 1.

    Shields

    1-Handed

    Riot shields, energy barriers, natural armor.

    FEA Feature: Parry

    The first W times you score a Win using a Defensive Move each Clash, deal 2 True Harm to the attacker.

    OFF Shield Bash

    DIE: 2 // PWR: 3 // DMG: 6 // EP: 0

    Impact.

    DEF Brace

    DIE: 6 // PWR: 2 // DMG: 0 // EP: 0

    On Win: Apply 1 Tremor to Self.

    DEF Redirect

    DIE: 3 // PWR: 3 // DMG: 0 // EP: 1

    Intercept.

    Ranged Weapon Shells

    Handgun

    1-Handed

    Pistols, revolvers, and submachine guns.

    FEA Feature: Quick Action

    Your first W Moves in a round cost -1 EP, to a minimum of 0.

    OFF Flurry

    DIE: 8 // PWR: -1 // DMG: 2 // EP: 1

    If this Move cost 0 EP, it gains Afflict Tremor 2.

    OFF Deadeye

    DIE: 2 // PWR: 4 // DMG: 9 // EP: 1

    If this Move cost 0 EP, its DMG is increased to 12.

    DEF Tactical Roll

    DIE: 5 // PWR: 2 // DMG: 0 // EP: 1

    If this Move cost 0 EP, it may be used again for 0 EP. This effect may only trigger once per round.

    Rifle

    2-Handed

    Automatic assault rifles, marksman rifles, burst-fire rifles.

    FEA Feature: Precise

    If a Move costed EP and scored at least 1 Win, it deals an additional 5 x W Harm, added before armor.

    OFF Aimed Shot

    DIE: 2 // PWR: 5 // DMG: 10 // EP: 1

    Reinforced

    OFF Center Mass

    DIE: 3 // PWR: 3 // DMG: 6 // EP: 0

    No additional effects.

    DEF Counter-Shot

    DIE: 1 // PWR: 5 // DMG: 2 // EP: 1

    On Break: -2 PWR on target's next attack.

    Support Weapons

    2-Handed

    Flamethrowers, chemical-sprayers, light machine guns.

    FEA Feature: Suppression

    Whenever you use a Move with Area, you may choose to do 0 Harm to all targets. If you do, they gain 6 x W stacks of Shock.

    OFF Pin Down

    DIE: 6 // PWR: -1 // DMG: 2 // EP: 1

    Afflict Shock 2.

    OFF Spray

    DIE: 6 // PWR: 0 // DMG: 3 // EP: 0

    Area 2.

    DEF Covering Fire

    DIE: 6 // PWR: 0 // DMG: 2 // EP: 1

    Intercept.

    Shotguns

    2-Handed

    Pump-action shotguns, scatterguns, and blunderbusses.

    FEA Feature: Punch Through

    Ignore 2 x W AV on each target.

    OFF Blast

    DIE: 3 // PWR: 2 // DMG: 6 // EP: 1

    Flow 3

    OFF Slug

    DIE: 2 // PWR: 4 // DMG: 10 // EP: 1

    Double this weapon's W for the duration of this Move.

    DEF Duck and Roll

    DIE: 4 // PWR: 1 // DMG: 0 // EP: 0

    Flow 2

    Cannons

    2-Handed

    Anti-material rifles, rocket launchers, and railguns.

    FEA Feature: Siege Engine

    All non-Innate Moves gain Bulwark 3 x W.

    OFF Bunker Buster

    DIE: 1 // PWR: 6 // DMG: 25 // EP: 2

    Vicious

    OFF Heavy Caliber

    DIE: 2 // PWR: 4 // DMG: 12 // EP: 1

    On Win: Reduce target's AV by 3 for the next Offensive Move targeting them.

    DEF Hunker Down

    DIE: 2 // PWR: 4 // DMG: 0 // EP: 1

    On Break: gain Hidden.

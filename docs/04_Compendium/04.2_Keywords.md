## Afflict X \[Status\]
Apply $X$ stacks of \[Status\] on Win.
## Area X
Select up to $X$ additional valid targets. Roll for the Clash one time. Each target rolls against your result. Text that says "target" reads "targets" for the purposes of this Move. Area on Offensive Moves only.
## Breach
The target's current Armor Value is halved for this round.
## Bulwark X
Ignore the effects and Harm of up to $X$ Unopposed dice the opponent controls.
## Deflect
If this Move has more Dice than the opposing Move, you are immune to Breaks.
## Flow X
On one or more Wins in a single Clash, gain X EP.
## Impact
If this Move deals 0 Harm, the target loses 1 EP. If the target has no EP, they take 5 true Harm instead.
## Innate
This Move relies on internal power (biology, psionics, code) rather than an external tool. It does not benefit from Weapon Features and does not use Weapon Tier for Scaling. It uses Anomaly Tier for Scaling and can be used while Disarmed.
## Intercept
This Move may be used in response to another Move not targeting you. If you do, you become that Move’s target.
## Reinforced
When you are Broken, only remove the next die in the sequence rather than all subsequent dice.
## Seeking
This Move ignores the [Hidden](04.1_Statuses_and_Conditions.md#hidden) Condition.
## Vicious
If this Move causes a Break, add +10 Harm to the total.

# Weapon Shells

!!!note "`W` values"
	In this document, wherever you see `W`, it represents the number of weapons of that shell you're wielding. For example, if you're wielding two handguns, `W` is equal to 2 for all Handgun abilities.

## Ranged Weapon Shells
### Handgun
*Pistols, revolvers, and submachine guns*
#### 1-Handed
#### Feature: Quick Action
!!!feature "Weapon Shell Feature"
	Your first *W* Moves in a round cost -1 EP, to a minimum of 0.
#### Flurry
!!! offensive "**:material-dice-6: DIE:** 8 // **:material-lightning-bolt: PWR:** -1 // **:fontawesome-solid-burst: DMG:** 2 // **:material-poker-chip: EP:** 1"
    If this Move cost 0 EP, it gains **Afflict Tremor 2**.
#### Deadeye
!!! offensive "**:material-dice-6: DIE:** 2 // **:material-lightning-bolt: PWR:** 4 // **:fontawesome-solid-burst: DMG:** 9 // **:material-poker-chip: EP:** 1"
    If this Move cost 0 EP, its :fontawesome-solid-burst: DMG is increased to 12.

!!! defensive "**:material-dice-6: DIE:** 5 // **:material-lightning-bolt: PWR:** 2 // **:fontawesome-solid-burst: DMG:** 0 // **:material-poker-chip: EP:** 1"
    If this Move cost 0 EP, it may be used again for 0 EP. This effect may only trigger once per round.

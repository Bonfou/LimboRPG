# Nitro
>He burned so bright, but burnt out so quickly.

A Nitro excels at using their own body for fuel, sacrificing HP for potent effects. Generate Heat upon taking Harm to cash in for rewards.

## Core Features

{% set feat_text %}
**Torch X:** Suffer X True Harm to activate a Move’s Torch effect. If a Move has multiple Torch costs, you must activate them individually. On **[Innate](../04_Compendium/04.2_Keywords.md#innate)** Moves, you may Torch 5 to increase a Move’s DMG by 50%, rounded down. 

**Heat Stacks:** Whenever you take Harm from any source, gain 1 Heat (max 6). Spend 3 Heat to reduce the EP cost of your next Move by 1, to a minimum of 0.
{% endset %}

# Operator
*The Weaponry Specialist*

## Core Features
### Momentum
!!! feature "FEATURE"
	Whenever you deal Harm to an opponent's HP, gain 1 Momentum (max 6). Momentum doesn't decay naturally during combat, but it clears at the end of combat.
	
	**Setup X:** Moves with this Keyword allow you to spend X Momentum on them to activate the associated effect.
